## 格式说明

采用简化 [MADR](https://adr.github.io/madr/) 格式，每篇包含四个部分：

- **背景** — 当时面临什么问题、有什么约束
- **决策** — 做了什么选择、具体怎么实现的
- **备选方案** — 考虑过哪些其他方案、为什么没选
- **后果** — 正面收益、负面影响、回头看得到的经验
